

## Install dependencies
```sh
yarn install
```

## Development
Development mode:
```sh
yarn dev
```
Run tests:
```sh
yarn test
```

## Deployment

```sh
yarn build
yarn start
```
Outputs:
- `build/`
- `public/build/`
